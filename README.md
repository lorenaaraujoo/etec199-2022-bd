- *Estrutura de Decisão*: IF, ELSE, ELSE-IF, CASE;
- *Estrutura de Repetição*: WHILE;
- *Identificação de Exceções*: TRY, CATCH;
- Funções de Agregação, Temporais e Conversão/Cast;
- JOINS;
- SUBQUERY;
- VIEW;
- PROCEDURE;
- TRIGGER;
- FUNCTION;
